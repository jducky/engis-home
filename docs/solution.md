ㅇ ENGIS Knowledge
- LLM 기반 자연어 질의응답 시스템으로 기업 내부 문서를 활용하여 사용자 질문에 대해 정확한 정보를 제공하는 AI 챗봇
- 주요 기능 : 문서 기반 질문 응답, 자연어 검색, 지식 관리 시스템 연동, FAQ 자동 생성
- 활용 분야 : 기업 지식관리 시스템, 고객 상담 챗봇, 내부 업무 지원 챗봇, 데이터 분석 질의 인터페이스

ㅇ ENGIS Insight
- 자연어 질문을 통해 데이터 분석을 수행할 수 있는 AI 기반 데이터 분석 솔루션임. 사용자는 SQL이나 분석 도구를 직접 사용하지 않고도 자연어 질문을 통해 데이터 조회 및 분석 결과를 확인할 수 있음
- 주요 기능 : 자연어 기반 데이터 조회, SQL 자동 생성, 데이터 분석 자동화, 분석 결과 요약 및 설명 생성
- 활용 분야 : 기업 데이터 분석, BI 시스템 보조 분석, 로그 데이터 분석, CSV / DB 데이터 질의 인터페이스

ㅇ ENGIS Vision
- 다양한 문서 이미지에서 텍스트를 자동 인식하는 AI 기반 문서 처리 솔루션임
- 주요 기능 : 스캔 문서 OCR, PDF 문서 텍스트 추출, 문서 자동 데이터 추출, 대량 문서 자동 처리
- 활용 사례 : 행정 문서 디지털화, 문서 검색 시스템, 데이터 입력 자동화, 문서 아카이빙 시스템
