Physics-Informed Neural Network (PINN) 구조
📐 Loss 함수 설계
총 손실함수는 데이터 손실과 물리 제약 손실의 가중합으로 구성됩니다. λ₁·λ₂는 물리 제약 강도를 조절하는 하이퍼파라미터입니다.
L_total = L_data
         + λ₁ × L_physics
         + λ₂ × L_boundary

현재: λ₁=0.8, λ₂=0.3
L_total = 0.0042 ✓
🌊 해양 물리 제약 조건
해양 물리 방정식을 위반하는 예측에 페널티를 부과합니다. 데이터가 부족한 구간에서도 물리적으로 타당한 예측을 보장합니다.
① 열역학: ∂T/∂t + u·∇T = κ∇²T
② 연속방정식: ∇·u = 0
③ DO 소모: ∂O₂/∂t = -R·f(T,S)
④ MLD: Ri < Ri_cr → 혼합
⑤ 수온-염분 관계: T-S 제약
🔄 자동 미분 (AD) 적용
PyTorch autograd를 활용해 편미분 항을 자동으로 계산합니다. 수치 미분 대비 정확도가 높고 역전파와 통합이 용이합니다.
# PyTorch autograd
T = model(x, t)
dT_dt = grad(T, t)
dT_dx = grad(T, x)
L_phys = MSE(
  dT_dt + u*dT_dx,
  kappa*laplacian(T)
)
