# ENGIS  
## AI · GIS · Data Service Capability

공공·해양·공간정보 분야에서  
**데이터 플랫폼 / 분석 서비스 / AI 적용 시스템**을 기획·구축합니다.

<div class="small">Capability Deck</div>

---

# 1. 회사 역량 개요

- **AI Applied Service**
  - 업무지원 챗봇
  - 문서분석 / OCR
  - 예측모델 / 설명형 분석

- **GIS / Spatial Platform**
  - 공간데이터 통합
  - 지도기반 분석
  - 경량 디지털트윈 서비스

- **Data / API Platform**
  - 데이터 표준화
  - Open API
  - 대시보드 및 운영체계

---

# 2. 차별화 포인트

<div class="box">

### 도메인 이해 기반 제안
공공기관·해양·연안·공간정보 업무의 실제 운영 방식까지 고려하여  
기술이 아닌 **업무 적용 중심**으로 제안합니다.

</div>

<div class="box">

### 기술 결합형 실행 역량
AI, GIS, 데이터 처리, API, UI를 분리하지 않고  
**하나의 서비스 구조**로 통합 설계·구현합니다.

</div>

---

# 3. 수행 방식

### 단계별 접근

1. **진단**
   - 현황 분석
   - 요구사항 정리
   - 데이터 검토

2. **설계**
   - 서비스 구조 설계
   - 기능 정의
   - 화면/UI 구상

3. **구축**
   - 플랫폼 개발
   - AI 모델 적용
   - API / 대시보드 구현

4. **확산**
   - 교육
   - 운영 안정화
   - 대외 서비스 확장

---
